# PrepMatrix: AI-Enhanced MERN Interview Platform


A full-stack interview preparation platform using the MERN stack, which helps students share and view real interview experiences, filtered by company and role.
Helps the users prepare for an upcoming interview by fetching experiences and an AI-powered custom interview guide (using Groq API) (Groq API) that
provides systematic, real-time insights on tech stacks, frequent coding problems, and recruitment patterns.

## Tech Stack :

- **Frontend:** React.js, Tailwind CSS, DaisyUI
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (with Mongoose)
- **AI Engine:** Groq API
- **Authentication:** JWT (JSON Web Tokens) with HTTP-Only Cookies
